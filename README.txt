Proyecto elaborado en Angular, Node y SQL.
Modulos a instalar: 

npm i express
npm i sha1 (encriptacion de passwords)
npm i @hapi/joi (validaciones)
npm i mysql (conexion con mysql)
npm i knex (querys simplificadas de sql SQL query builder)
npm i dotenv (variables de entorno)
npm i jsonwebtoken (creacion de tokens basados en JSON)
npm i uuid (creacion de ids unicos)
npm i nodemailer (envio de mails)

Arranque del servidor: nodemon npm start
Arranque del cliente: ng serve -o
